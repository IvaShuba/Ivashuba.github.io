Smart Greenhouse – Master/Slave MCU System

Embedded greenhouse control system built on a two‑node architecture: Master (Arduino Uno) and Slave (ATmega328PB). Implements SPI communication, sensor fusion with EMA filtering, actuator control, LCD interface, and low‑power operation with sleep modes.

• SPI master–slave protocol with 8 command types
• EMA‑filtered temperature, humidity, and soil moisture
• Deep‑sleep slave node with interrupt wake‑up
• LCD (I2C) interface and UART JSON output
• Servo‑controlled ventilation, pump & fan automation
• Structured SensorData package with checksum
